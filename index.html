Class in HTML

  1. There can be multiple class in a single tage.
  2. In css class can be accessed by dot ( . ).
  3. Class is mainly use for Styling (We can use it with JavaScript too but not recomended)

Id in HTML

  • Id must be unique in a single tag.
  • In css id can be accessed by Hash ( # ).
  • Id is mainly use for identify a tag via JavaScript. (We can also use for CSS).

The difference between ID and Class of HTML

S.N. CLASS ID
1 There can be multiple class in a single tage. Id must be unique in a single tag.
2 In css class can be accessed by dot ( . ). In css id can be accessed by Hash ( # ).
3 Class is mainly use for Styling (We can use it with JavaScript too but not recomended) Id is mainly use for identify a tag via JavaScript. (We can also use for CSS).
